# 🍽️ RouxMind — Where Every Plate Tells Your Story

AI-powered recipe generator that turns your ingredients into delicious recipes instantly.

## ✨ Features

- 📸 Smart Image Recognition — Upload photo, AI detects ingredients
- ⚡ Ultra-Fast Generation — Recipes in seconds
- 🎯 Personalized Preferences — Spice, servings, time
- 🌐 20+ Languages Supported
- 📱 Mobile-First Design
- 📊 Analytics Dashboard
- 🎨 Premium Glass-morphism UI
- 📄 PDF Export
- 💾 Save & Share Recipes

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Mesh API key ([Get free key](https://meshapi.ai))

### Installation

```bash
# Clone
git clone https://github.com/yourusername/rouxmind.git
cd rouxmind

# Install
npm install

# Setup env
cp .env.example .env
# Edit .env with your API keys

# Run
npm start
