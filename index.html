✦ Protocol — Sheets × Mesh

Everything you need to know about the engine

We send base64 image + prompt to google/gemini-2.5-flash-lite via meshapi.ai/v1/chat/completions at $0.10/$0.40 per 1M tokens. It returns dish name only. 20s timeout, fallback to user typed dishName if fails. Confidence 92-98%.

Logs!A:I (timestamp, user, dish, servings, spice, status, tokens, cost, time) is append-only audit. Users!A:F (name, total, streak, lastActive ISO, rank, xp) is gamification store. Global!A:D (date, total, users, topDish) is daily rollup, cached 5min in memory to save quota.

Streak: midnight logic — same day keep, yesterday +1, else reset 0 → 1. XP = total×3 + streak×7 + spice×2. Ranks: 0 Beginner 👶, 10 Line Cook 🥉, 50 Sous Chef 🥈, 100 Executive Chef 🥇, 500 Grand Master 👑. Leaderboard sortable by total/streak/xp, limit 50.

Images are converted to data:image/... base64 in browser and sent to Mesh API in-memory, never persisted on Vercel or Sheets. Sheets gets only dish name, not image. Rate limit 40 req/min per IP. CORS open but Sheets creds stay server-side only.

Built by Sooban Talha Technologies — Karachi to global. Cheapest model on Mesh is inclusionai/ling-2.6-flash $0.01/$0.03 (ULTRA_FAST), then Gemma-3-4B $0.04/$0.08 (TEXT), then Gemini 2.5 Flash Lite $0.10/$0.40 (VISION). All via same endpoint.
